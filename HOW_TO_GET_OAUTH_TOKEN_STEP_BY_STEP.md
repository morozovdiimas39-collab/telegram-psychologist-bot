# Как получить OAuth токен на oauth.yandex.ru

## Пошаговая инструкция

### Вариант 1: Использовать существующее приложение

1. **На странице https://oauth.yandex.ru/** (где ты сейчас)
2. **Найди любое приложение** из списка (например, "CleanerRSYA" или "WS")
3. **Нажми на три точки** (⋮) справа от приложения
4. **Выбери "Получить токен"** или "Токен доступа"
5. **Скопируй токен** — это и есть `YANDEX_CLOUD_TOKEN`

### Вариант 2: Создать новое приложение

1. **На странице https://oauth.yandex.ru/**
2. **Нажми кнопку "+ Создать"** (справа вверху)
3. **Заполни форму:**
   - Название: `Deployer` (или любое другое)
   - Платформа: выбери "Веб-сервис" или "Другое"
   - Описание: опционально
4. **Сохрани приложение**
5. **После создания:**
   - Открой приложение
   - Найди раздел "Токен доступа" или "Access Token"
   - Скопируй токен

### Вариант 3: Получить токен через API (если есть права)

Если у тебя есть Client ID и Client Secret приложения:

1. Открой любое приложение из списка
2. Скопируй **Client ID** и **Client Secret**
3. Используй их для получения токена через API

Но проще всего использовать **Вариант 1** или **Вариант 2**!

## Что делать с токеном

После получения токена:

1. **Скопируй токен** (длинная строка букв и цифр)
2. **Добавь его в функцию `setup-database`:**
   - Открой функцию в Yandex Cloud Console
   - Версии → Редактировать
   - Переменные окружения → Добавить:
     - **Ключ**: `YANDEX_CLOUD_TOKEN`
     - **Значение**: твой токен
3. **Сохрани версию**

## Важно

- ⚠️ Токен может быть длинным — скопируй его полностью
- ⚠️ Токен действует ограниченное время (обычно 1 год)
- ⚠️ Если токен истёк — получи новый

## Если не видишь кнопку "Получить токен"

1. Попробуй открыть приложение (кликни на название)
2. Или создай новое приложение через "+ Создать"
3. В настройках приложения должен быть раздел с токенами
