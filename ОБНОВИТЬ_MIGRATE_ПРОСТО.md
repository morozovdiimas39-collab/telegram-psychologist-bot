# Обновить migrate — 3 шага

Код уже исправлен. Нужно только загрузить его в облако.

---

## Шаг 1. Открой функцию

1. Зайди на https://console.cloud.yandex.ru/functions
2. Найди функцию **migrate** (или по ID `d4eoo7gt252g039lr5bj`)
3. Открой её

---

## Шаг 2. Обнови код

1. Перейди во вкладку **«Редактор»**
2. Открой файл `index.py` в проекте (в папке `backend/migrate/index.py`)
3. Скопируй **весь** код из `index.py`
4. Вставь в редактор в браузере (заменив то, что там есть)
5. Убедись, что в `requirements.txt` указано:
   ```
   psycopg2-binary>=2.9.0
   requests>=2.28.0
   ```

---

## Шаг 3. Создай версию

1. Нажми кнопку **«Создать версию»**
2. Подожди 1–2 минуты
3. Готово

---

## Проверка

Зайди на страницу **Миграции** в своём приложении и нажми «Применить миграции». Ошибка должна исчезнуть.
