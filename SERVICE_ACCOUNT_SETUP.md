# Как использовать ключ сервисного аккаунта

## ⚠️ Важно!

У тебя есть **идентификатор и секретный ключ**, но для работы с Yandex Cloud API нужен **OAuth токен пользователя**, а не ключ сервисного аккаунта напрямую.

## Что делать

### ✅ Используй OAuth токен (РЕКОМЕНДУЕТСЯ)

1. **Открой:** https://oauth.yandex.ru/
2. **Войди** в свой аккаунт Yandex
3. **Нажми** "Получить токен" или "Разрешить"
4. **Разреши** доступ к Yandex Cloud
5. **Скопируй** токен — это и есть `YANDEX_CLOUD_TOKEN`

### ❌ Ключ сервисного аккаунта

Ключ сервисного аккаунта (`YCAJEdMV0k0HRtrZNmsNI3NNS` + секрет) используется для других целей (например, через Yandex Cloud SDK), но не напрямую в функциях через API.

**Для функций деплойера нужен OAuth токен пользователя!**

## Как добавить в функцию

1. Открой функцию `setup-database` в [Yandex Cloud Console](https://console.cloud.yandex.ru/functions)
2. Версии → последняя версия → Редактировать
3. Переменные окружения → Добавить:
   - **Ключ**: `YANDEX_CLOUD_TOKEN`
   - **Значение**: твой OAuth токен из https://oauth.yandex.ru/
4. Сохрани версию

## Готово!

После этого функция сможет создавать VM через Yandex Cloud API.
