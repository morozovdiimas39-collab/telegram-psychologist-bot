# Как открыть сайт по crode.ru вместо IP

Сайт работает по IP — значит nginx настроен. Осталось настроить DNS.

---

## Шаг 1. Узнай IP сервера

В деплойере: зайди на страницу Deploy → в карточке конфига с crode.ru указан IP сервера.  
Или посмотри в логах последнего деплоя.

---

## Шаг 2. Добавь A-запись в DNS

Зайди в панель управления доменом (Reg.ru, Timeweb, Cloudflare, Namecheap и т.п.).

**Добавь A-запись:**

| Тип | Имя | Значение | TTL |
|-----|-----|----------|-----|
| A   | @   | IP_АДРЕС_СЕРВЕРА | 300 |

**Если нужен www.crode.ru:**

| Тип | Имя | Значение | TTL |
|-----|-----|----------|-----|
| A   | www | IP_АДРЕС_СЕРВЕРА | 300 |

*Имя `@` = корень домена (crode.ru), `www` = www.crode.ru*

---

## Шаг 3. Подожди 5–30 минут

DNS обновляется не мгновенно. Проверь: `ping crode.ru` — должен показать IP сервера.

---

## Шаг 4. Перезапусти деплой (для SSL)

Когда DNS заработает, нажми «Задеплоить фронтенд» ещё раз. certbot автоматически выпустит SSL-сертификат, и сайт откроется по **https://crode.ru**.

---

## Где настраивать DNS

- **Reg.ru** → Домены → Управление → Зона DNS
- **Timeweb** → Домены → Управление → DNS-записи
- **Cloudflare** → DNS → Records
- **Namecheap** → Domain List → Manage → Advanced DNS
