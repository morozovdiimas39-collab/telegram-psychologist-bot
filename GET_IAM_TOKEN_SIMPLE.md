# Как получить IAM токен для Yandex Cloud

## ✅ Самый простой способ

### Шаг 1: Открой Yandex Cloud Console

Перейди по ссылке: https://console.cloud.yandex.ru/

### Шаг 2: Получи IAM токен

1. **В правом верхнем углу** нажми на свой аккаунт (аватар или имя)
2. В выпадающем меню найди **"Получить IAM токен"** или **"Создать токен"**
3. Нажми на эту опцию
4. **Скопируй токен** — это длинная строка букв и цифр

### Шаг 3: Используй токен

Этот токен и есть `YANDEX_CLOUD_TOKEN` для функций!

## Где добавить токен

1. Открой функцию `setup-database` в Yandex Cloud Console
2. Версии → Редактировать
3. Переменные окружения → Добавить:
   - **Ключ**: `YANDEX_CLOUD_TOKEN`
   - **Значение**: твой IAM токен
4. Сохрани версию

## Про ключ сервисного аккаунта

Ключ сервисного аккаунта (`YCAJEaH6MqgeFbyCVE9XGoQoN` + секрет) можно использовать для других задач, но для функций деплойера проще использовать **IAM токен из консоли**.

## Важно

- ✅ IAM токен работает сразу
- ✅ Не требует дополнительных библиотек
- ✅ Действует ограниченное время (обычно 12 часов)
- ✅ Можно получить новый токен в любой момент
