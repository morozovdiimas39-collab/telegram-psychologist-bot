# Как получить токен для Yandex Cloud API

## ⚠️ Важно понимать

**oauth.yandex.ru** — это для OAuth приложений Яндекс ID (Диск, Почта и т.д.), **НЕ для Yandex Cloud API**.

Для работы с **Yandex Cloud API** нужен **IAM токен**, который получается из **Yandex Cloud Console**.

## ✅ Правильный способ

### Способ 1: IAM токен из Yandex Cloud Console (РЕКОМЕНДУЕТСЯ)

1. **Открой:** https://console.cloud.yandex.ru/
2. **Войди** в свой аккаунт Yandex Cloud
3. **В правом верхнем углу** нажми на свой аккаунт (аватар или имя)
4. **Выбери:** "Получить IAM токен" или "Создать токен"
5. **Скопируй токен** — это и есть `YANDEX_CLOUD_TOKEN`

**Этот токен работает для всех функций Yandex Cloud API!**

### Способ 2: Ключ сервисного аккаунта

Если у тебя есть ключ сервисного аккаунта:
- Идентификатор: `YOUR_KEY_ID`
- Секрет: `YOUR_SECRET_KEY` *(из консоли Yandex Cloud)*

Но для функций нужен **IAM токен**, который можно получить из ключа сервисного аккаунта через API, но это сложнее.

**Проще использовать Способ 1!**

## Что делать с токеном

После получения IAM токена:

1. Открой функцию `setup-database` в [Yandex Cloud Console](https://console.cloud.yandex.ru/functions)
2. Версии → Редактировать
3. Переменные окружения → Добавить:
   - **Ключ**: `YANDEX_CLOUD_TOKEN`
   - **Значение**: твой IAM токен из консоли
4. Сохрани версию

## Резюме

- ❌ **oauth.yandex.ru** — для Яндекс ID приложений (Диск, Почта)
- ✅ **console.cloud.yandex.ru** — для Yandex Cloud API (VM, функции)
- ✅ **Используй IAM токен** из Yandex Cloud Console
