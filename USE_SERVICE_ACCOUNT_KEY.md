# Как использовать ключ сервисного аккаунта

## У тебя есть ключ сервисного аккаунта

**Идентификатор ключа:** `YOUR_KEY_ID`  
**Секретный ключ:** `YOUR_SECRET_KEY` *(вставь свой из консоли Yandex Cloud)*

## ⚠️ Важно

Для функций Yandex Cloud нужен **IAM токен**, а не ключ сервисного аккаунта напрямую.

Ключ сервисного аккаунта используется для получения IAM токена через API, но это сложнее.

## ✅ Рекомендуемый способ

**Используй IAM токен из Yandex Cloud Console:**

1. Открой https://console.cloud.yandex.ru/
2. В правом верхнем углу нажми на свой аккаунт
3. Выбери "Получить IAM токен" или "Создать токен"
4. Скопируй токен

**Это проще и работает сразу!**

## Альтернатива: Использовать ключ сервисного аккаунта

Если хочешь использовать именно этот ключ, нужно:

1. **Получить полный JSON ключ** (не только ID и секрет, а JSON файл с приватным ключом)
2. **Использовать его для создания JWT токена**
3. **Получить IAM токен из JWT**

Но это сложнее и требует дополнительных библиотек.

## Что делать сейчас

**Проще всего:**
1. Получи IAM токен из консоли (правый верхний угол → "Получить IAM токен")
2. Используй его как `YANDEX_CLOUD_TOKEN` в функции

Ключ сервисного аккаунта можешь сохранить для других задач, но для функций деплойера используй IAM токен из консоли.
