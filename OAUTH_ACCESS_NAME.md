# Название доступа для OAuth приложения

## Что указать в поле "Название доступа"

Для работы функций деплойера с Yandex Cloud API укажи:

### Вариант 1 (рекомендуется):
```
Yandex Cloud API
```

### Вариант 2:
```
cloud-api
```

### Вариант 3:
```
Yandex Cloud
```

## Что это даёт

Этот доступ позволит приложению:
- ✅ Создавать и управлять VM (виртуальными машинами)
- ✅ Создавать и управлять облачными функциями
- ✅ Работать с ресурсами Yandex Cloud через API

## После создания приложения

1. **Сохрани приложение**
2. **Скопируй Client ID и Client Secret** (если они будут показаны)
3. **Используй их для получения токена** или получи токен через Yandex Cloud Console

## Альтернатива (проще)

Если не хочешь создавать OAuth приложение, используй **IAM токен** из Yandex Cloud Console:
1. Открой https://console.cloud.yandex.ru/
2. В правом верхнем углу → "Получить IAM токен"
3. Скопируй токен

Это проще и работает сразу!
