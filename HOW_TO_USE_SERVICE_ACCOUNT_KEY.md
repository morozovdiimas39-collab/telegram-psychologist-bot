# Как использовать ключ сервисного аккаунта

## У тебя есть ключ сервисного аккаунта

**Идентификатор ключа:** `YOUR_KEY_ID`  
**Секретный ключ:** `YOUR_SECRET_KEY` *(из консоли Yandex Cloud)*

## Два способа использования

### Способ 1: Использовать OAuth токен (проще)

1. Открой https://oauth.yandex.ru/
2. Получи OAuth токен пользователя
3. Используй его как `YANDEX_CLOUD_TOKEN`

**Это проще и работает сразу!**

### Способ 2: Использовать ключ сервисного аккаунта

Для использования ключа сервисного аккаунта нужно добавить **две** переменные окружения:

- `YANDEX_SERVICE_ACCOUNT_KEY_ID` = `YOUR_KEY_ID`
- `YANDEX_SERVICE_ACCOUNT_KEY_SECRET` = `YOUR_SECRET_KEY`

Но для этого нужно обновить код функции, чтобы она поддерживала ключи сервисного аккаунта.

## Рекомендация

**Используй OAuth токен** (способ 1) — это проще и не требует изменений в коде.

Если всё же хочешь использовать ключ сервисного аккаунта, нужно:
1. Обновить код функции для поддержки ключей
2. Добавить библиотеку для работы с JWT токенами
3. Это сложнее и требует больше изменений

## Что делать сейчас

1. Открой https://oauth.yandex.ru/
2. Получи OAuth токен
3. Используй его как `YANDEX_CLOUD_TOKEN` в функции `setup-database`

Это самый простой способ!
